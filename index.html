I've decided to put together a personal site in order to throw some of my development thoughts up somewhere as well as offer a diversion from the problems I've been working through in Cracking the Coding Interview 6th Edition. When I started this project, it had been a few weeks straight of putting in time every day working through problems in the book. I do best when I'm working on a variety of things so it was about time to tackle something completely different.

If you're interested in the tech that went into this site, you can refer to the about the site page. There's the résumé page which is self explanatory. In the future, I hope to add a blog component, but I want to try to write that portion myself since it's not a particularly pressing need and it should be a nice project to tackle.

As it turns out, DigitalOcean doesn't offer email routing for droplets so at the moment, I'm not listing a contact address. For now, please contact me on GitHub or LinkedIn - links to these can be found in the footer of any page.
